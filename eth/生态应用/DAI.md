# DAI / USDS

## 概念简介

DAI是由MakerDAO协议发行的去中心化稳定币，通过超额抵押加密资产的方式生成，锚定美元保持1:1的价值比例。作为以太坊生态中最重要的去中心化稳定币，DAI完全运行在智能合约上，无需依赖中心化机构或法币银行账户，代表了加密货币领域对去中心化金融基础设施的探索。

MakerDAO由Rune Christensen于2014年创立，2017年12月在以太坊主网正式上线。与USDT、USDC等由中心化公司发行并依赖法币储备的稳定币不同，DAI通过智能合约自动化管理，用户抵押ETH、WBTC等加密资产即可铸造DAI，所有操作透明可验证，体现了真正的去中心化精神。

2024年8月，MakerDAO进行了品牌重塑，更名为Sky Protocol，DAI也相应升级为USDS（Sky Dollar）。尽管名称变化，核心机制保持不变，DAI与USDS可以1:1互换，现有DAI持有者可选择是否升级。截至2024年，DAI/USDS的市值约为50-60亿美元，是仅次于USDT和USDC的第三大稳定币，也是市值最大的去中心化稳定币。

## 核心特性

**超额抵押铸造机制**

DAI采用超额抵押模型，用户需要存入价值高于铸造金额的加密资产作为抵押品。例如，用户存入价值1000美元的ETH，根据抵押率设置（通常为150%），最多可以铸造约666美元的DAI。这种超额抵押确保了即使抵押品价格下跌，系统仍有足够的价值支撑所有流通中的DAI。

用户通过创建"金库"（Vault，原称CDP/抵押债务头寸）来铸造DAI。金库是一个智能合约，记录了用户的抵押品和债务。用户可以随时追加抵押品、偿还DAI关闭金库、或提取部分抵押品。这种机制给予用户极大的灵活性和控制权。

**多元化抵押品支持**

MakerDAO支持多种加密资产作为抵押品，包括：
- **ETH（以太坊）**：最主要的抵押品，占比最大
- **WBTC（比特币包装代币）**：将比特币价值引入以太坊
- **USDC、USDP等中心化稳定币**：用于稳定DAI的锚定
- **流动性质押代币**：如stETH、rETH等，既抵押又赚取质押收益
- **其他ERC-20代币**：经过治理投票批准的优质资产

每种抵押品都有独立的风险参数，由MKR持有者通过治理投票决定，包括抵押率、稳定费率、债务上限等。这种多元化降低了单一资产风险，增强了系统稳定性。

**稳定费与DAI储蓄率**

- **稳定费（Stability Fee）**：用户铸造DAI需要支付的年化利息，类似于借贷利率。稳定费由MKR持有者投票决定，根据市场供需调整。当DAI价格低于1美元时，提高稳定费以减少供应；反之则降低。稳定费收入进入协议金库。

- **DAI储蓄率（DSR）**：DAI持有者可以将DAI锁入DSR合约赚取利息，年化收益率由治理决定。DSR用于调节DAI需求——提高DSR吸引用户持有DAI，有助于维持价格稳定。DSR的资金来源于稳定费收入。

**自动化清算系统**

当金库的抵押率低于清算线（如130%）时，系统会触发自动清算。清算人（Keeper）可以购买被清算的抵押品，支付相应的DAI偿还债务，并获得清算折扣（通常为3-13%）作为激励。这种机制确保系统始终有足够的抵押品支撑DAI。

清算过程完全自动化，任何人都可以运行清算机器人参与。如果清算收入不足以覆盖债务，协议会通过债务拍卖增发MKR代币来填补缺口，这为MKR持有者创造了风险激励——他们有动力维护系统健康以避免MKR增发稀释。

**MKR治理代币**

MKR是MakerDAO的治理代币，持有者拥有协议的投票权，可以决定：
- 新增或移除抵押品类型
- 调整各种风险参数（抵押率、稳定费、债务上限）
- 修改DSR利率
- 协议升级和重大决策

MKR还有经济价值捕获机制：稳定费收入用于从市场回购并销毁MKR，减少供应；若系统出现坏账，则增发MKR筹集资金，MKR持有者承担最终风险。

**Sky Protocol升级与USDS**

2024年Sky Protocol升级引入了多项改进：
- **USDS稳定币**：DAI的升级版，两者可1:1互换，逐步过渡
- **SKY代币**：MKR升级版（1 MKR = 24,000 SKY），增强流动性和治理参与度
- **Sky储蓄率（SSR）**：升级版DSR，提供更灵活的收益选项
- **Sky Token Stars**：基于美国国债的代币化产品，为用户提供稳定收益
- **子DAO模式**：将协议拆分为多个专注子DAO，提高治理效率

## 技术优势

**真正的去中心化**

DAI是完全去中心化运行的稳定币，所有规则编码在智能合约中，无需信任单一机构。Circle可以冻结USDC地址，Tether可以关闭账户，但没有任何实体能单方面控制DAI。这种去中心化使DAI具有抗审查性，任何人都可以铸造和使用。

**完全透明与可验证**

所有抵押品、债务、清算记录都在以太坊区块链上公开可查。用户可以实时验证每一枚DAI都有足够的抵押品支撑，无需信任第三方审计报告。这种透明度是中心化稳定币无法比拟的。

**无需许可与全球可用**

任何人只要有加密资产就可以铸造DAI，无需KYC认证、无需银行账户、无地域限制。这为全球无银行账户人群提供了获取稳定美元资产的途径。

**可编程与DeFi集成**

作为以太坊原生资产，DAI无缝集成到各种DeFi协议中。Aave、Compound、Uniswap、Curve等数百个协议支持DAI。用户可以用DAI进行借贷、交易、流动性挖矿、收益耕作等复杂金融操作。

**抗单点故障**

DAI的多元化抵押品和去中心化架构使其不受单一银行或资产风险影响。即使某个抵押品价格崩溃，其他抵押品仍能支撑系统。2023年USDC因SVB银行危机脱锚时，MakerDAO迅速调整风险参数，DAI保持了相对稳定。

## 发展历程

**2014-2017年：孕育与准备**

2014年，Rune Christensen提出MakerDAO概念，设计基于以太坊的去中心化稳定币系统。2015-2017年进行了多轮测试和开发，募集资金并建立社区。

**2017年12月：单抵押DAI（SAI）上线**

MakerDAO主网正式启动，推出第一版DAI（后称SAI），仅支持ETH作为抵押品。早期供应量较小，主要在DeFi早期采用者中流通。

**2018-2019年：早期增长**

DAI逐渐被Compound、Uniswap等早期DeFi协议采用，成为重要的稳定币选项。SAI供应量从数百万增长至数千万美元。

**2019年11月：多抵押DAI（MCD）升级**

MakerDAO推出多抵押DAI（Multi-Collateral DAI），支持多种资产作为抵押品，大幅提升了系统的灵活性和可扩展性。SAI逐步迁移至新版DAI。引入DSR储蓄率功能，允许DAI持有者赚取利息。

**2020年3月：黑色星期四**

全球金融市场因疫情崩盘，ETH价格单日暴跌50%，从约200美元跌至80美元。MakerDAO系统面临严重考验，大量金库被清算，以太坊网络拥堵导致清算机制短暂失效，系统出现约500万美元坏账。

MakerDAO通过债务拍卖增发MKR筹集资金填补缺口，这次危机暴露了系统弱点，但也证明了应急机制的有效性。事后MakerDAO改进了清算系统、引入了更多流动性抵押品（如USDC）、调整了风险参数。

**2020-2021年：DeFi Summer爆发**

DeFi生态繁荣，DAI需求激增。DAI供应量从黑色星期四后的约1亿美元暴涨至超过100亿美元（2021年峰值）。DAI成为DeFi生态的基础稳定币，在Uniswap、Curve、Aave等协议中占据重要地位。

**2022年：真实世界资产（RWA）引入**

为提高收益和稳定性，MakerDAO开始投资真实世界资产，包括美国国债、企业债券等。RWA占协议资产比例逐步提升，引发了关于去中心化纯粹性的社区讨论。

**2023年：市场调整**

随着DeFi市场降温和利率环境变化，DAI供应量从峰值回落至50-60亿美元区间。MakerDAO持续优化风险参数、引入新抵押品类型、扩大RWA投资。

**2024年8月：Sky Protocol品牌重塑**

MakerDAO正式更名为Sky Protocol，推出USDS和SKY代币。这次升级旨在简化用户体验、扩大协议规模、提升治理效率。DAI持有者可选择升级至USDS，两者可1:1互换，DAI将继续支持。

## 应用场景

**DeFi借贷协议**

DAI是Aave、Compound、Spark等借贷协议的核心资产之一。用户可以：
- 存入DAI赚取存款利息（通常3-8%年化）
- 以DAI作为抵押品借入其他资产
- 借入DAI进行杠杆操作或套利

**去中心化交易所（DEX）**

DAI在Uniswap、Curve、Balancer等DEX中提供重要的交易对和流动性：
- DAI/USDC、DAI/USDT等稳定币交易对提供低滑点兑换
- DAI/ETH、DAI/WBTC等交易对为用户提供稳定币入口
- 流动性提供者赚取交易手续费和流动性挖矿奖励

**稳定币价值存储**

追求去中心化和抗审查的用户选择持有DAI而非USDC/USDT，规避中心化稳定币的冻结风险。将DAI存入DSR/SSR赚取无风险收益，类似传统银行储蓄但完全链上化。

**跨境支付与汇款**

虽然不如USDT/USDC流行，但DAI在注重隐私和去中心化的跨境支付场景中有应用，特别是在传统金融服务受限的地区。

**DAO与协议金库**

许多DAO和DeFi协议使用DAI作为金库资产，因其去中心化特性和稳定性。DAI不受单一公司控制，适合长期持有。

**链上结算与支付**

一些Web3应用和NFT市场支持DAI支付，为用户提供去中心化的结算选项。

## 风险与挑战

**复杂度与用户门槛**

相比"存1美元得1稳定币"的简单模型，DAI的超额抵押、清算、治理等机制较为复杂。普通用户理解门槛高，限制了大规模采用。铸造DAI需要管理金库、监控抵押率、应对清算风险，不如直接购买USDC便捷。

**抵押品价格波动风险**

DAI依赖加密资产抵押，这些资产价格波动性高。极端市场波动可能导致大规模清算，甚至系统坏账。虽然黑色星期四后系统得到改进，但类似风险仍然存在。

**资本效率低**

超额抵押模型意味着资本效率低。用户需要锁定价值150%以上的资产才能铸造100%的DAI，远不如中心化稳定币的1:1效率。这限制了DAI的供应扩张速度。

**治理风险与中心化担忧**

尽管理论上去中心化，MakerDAO的治理实际参与度有限，大部分MKR集中在少数大户手中。治理提案通过率低，重大决策可能被少数人操控。RWA的引入也引发了对中心化的担忧——投资美国国债需要依赖传统金融机构和托管服务。

**扩展性挑战**

由于依赖以太坊主网和加密资产抵押，DAI的供应增长受限于市场对加密资产的需求和以太坊Gas成本。在熊市中，用户不愿抵押贬值资产，DAI供应收缩。

**与中心化稳定币的竞争**

USDC、USDT在流动性、用户认知、机构支持方面远超DAI。大多数用户优先选择更简单、更广泛支持的中心化稳定币。DAI的市值增长速度落后于整体稳定币市场。

**监管不确定性**

虽然DAI去中心化，但运营MakerDAO的实体和基金会仍可能受到监管压力。RWA投资使协议与传统金融系统产生关联，增加了监管风险。

## 未来展望

**Sky Protocol生态建设**

Sky Protocol的多DAO架构旨在提高治理效率和专业化。不同子DAO专注于RWA、流动性管理、协议增长等领域，协同推动生态发展。USDS品牌升级和Sky Token Stars等产品创新可能吸引更多用户。

**RWA整合深化**

MakerDAO/Sky将继续扩大真实世界资产投资，从美国国债扩展至企业债券、房地产、贸易融资等。RWA能提供稳定收益，减少对加密市场波动的依赖，但也需要平衡去中心化与效率。

**Layer2扩展**

在以太坊Layer2（如Arbitrum、Optimism、Base）和其他兼容链上部署DAI/USDS，降低使用成本，提高可访问性。跨链桥和流动性聚合将使DAI/USDS在多链生态中无缝流通。

**去中心化稳定币领导者**

作为最大的去中心化稳定币，DAI/USDS有机会巩固其在注重抗审查和隐私的用户群体中的地位。随着监管收紧，部分用户可能从中心化稳定币转向去中心化替代品。

**与CBDC的互补**

未来各国推出央行数字货币（CBDC）后，DAI可能与CBDC形成互补关系。DAI提供去中心化和可编程性，CBDC提供政府背书和法定地位，两者服务不同用户需求。

**治理与社区成熟**

随着Sky Protocol改革和社区教育的深入，治理参与度可能提升。更广泛的MKR/SKY分布和子DAO模式可能增强决策质量和执行效率。
