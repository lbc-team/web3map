# Shamir 密钥共享

## 概念简介

Shamir 密钥共享（Shamir Secret Sharing, SSS）是由 Adi Shamir 于 1979 年提出的(t, n)门限秘密共享方案。该方案基于拉格朗日插值法，通过构造一个 t-1 次多项式，将秘密作为常数项，每个份额（子秘密）是满足该多项式的一个坐标点。

门限秘密共享是一种将秘密 S 分成 n 份并分发给 n 个参与者的密码学技术。其中门限值 k（介于 1 到 n 之间）确保任何 k-1 份或更少的秘密份额无法计算出关于秘密 S 的任何信息，但任何 k 份或更多份额可以计算出秘密 S。

## 技术原理

**拉格朗日插值定理**：任意 t 个份额可以重构多项式以恢复秘密，而 t-1 个或更少的份额无法重构多项式，从而获取不到关于秘密的任何信息。

**数学基础**：核心数学原理是拉格朗日插值算法，使用由多项式生成的函数，秘密作为多项式的常数项，每个份额是该多项式在不同点的值。

## 在区块链中的应用

**去中心化密钥共享方案**：将密钥分成 n 份，对每个秘密部分加密后存储在区块链上，解决密钥分发和存储问题。当密钥恢复请求者需要恢复密钥时，可以向全网广播密钥恢复消息，并向密钥存储节点支付一定数量的代币。

**分布式存储**：密钥片段分布在区块链网络的不同节点上，需要从整个区块链网络中获取超过门限数量的密钥片段才能重构私钥。

## 应用场景

1. **多签[钱包](https://learnblockchain.cn/tags/%E9%92%B1%E5%8C%85)**：使用门限签名保护资产，需要多方协作才能动用资金
2. **密钥备份**：将私钥分片存储在不同位置，防止单点故障
3. **安全存储**：防止密钥被单一方完全掌握
4. **访问控制**：实现灵活的权限管理机制
