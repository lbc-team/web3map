# Ethena

## 概念简介

Ethena是一个建立在以太坊上的合成美元协议,通过delta中性策略提供链上、抗审查的稳定币USDe。与传统的超额抵押或算法稳定币不同,Ethena采用"现货+永续合约对冲"的方式维持稳定性,同时为用户提供高收益的stUSDe质押产品。

Ethena于2023年推出,由前PayPal和BitMEX高管Guy Young创立。协议的核心创新在于将加密原生的套利策略产品化,通过持有ETH/stETH现货并在衍生品市场开空单对冲,实现了稳定币的发行,同时捕获了资金费率和质押收益。

截至2024年,Ethena已成为增长最快的稳定币项目之一,USDe流通量达到数十亿美元,stUSDe提供了DeFi中最高的美元计价收益率之一(15-30%),吸引了大量用户和机构关注。

## 核心特性

### 1. USDe稳定币

合成美元,保持1:1锚定USD:
- 由质押的ETH/BTC和对冲仓位支持
- 链上透明,可审计
- 抗审查,无单点故障
- 可在DeFi生态自由流通

### 2. Delta中性策略

通过同时持有多空仓位维持稳定:
- 持有stETH等生息资产作为抵押品
- 在CEX开等量永续合约空单
- 价格波动被完全对冲
- 赚取质押收益+资金费率

### 3. stUSDe(质押USDe)

用户质押USDe获得收益:
- 自动复利的收益代币
- 收益来源于资金费率+质押奖励
- 高收益率(历史15-35% APY)
- 可用作DeFi抵押品

### 4. 多资产抵押

支持多种优质资产:
- stETH(Lido)
- mETH(Mantle)
- BTC(包装资产)
- 其他流动性质押代币

### 5. 机构级风险管理

- 多家顶级CEX作为对冲渠道
- 专业的资产托管方案
- 实时监控和再平衡
- 保险基金保护

### 6. 链上透明

所有抵押品和仓位可在链上验证,实时查看储备证明。

## 技术优势

### 1. 可扩展性

理论上可以扩展至数百亿美元规模,受限于衍生品市场深度。

### 2. 高收益

相比传统稳定币(0-5%),提供显著更高的收益率。

### 3. 资本效率

用户可以铸造USDe后质押为stUSDe,同时保持流动性和收益。

### 4. 抗审查

去中心化的抵押品和对冲策略,不依赖银行系统。

### 5. 可组合性

USDe和stUSDe可用于各类DeFi协议,释放更多收益机会。

## 工作原理

### USDe铸造流程

1. 用户存入stETH等抵押品
2. Ethena在CEX开对应的空单对冲
3. 用户收到等值USDe
4. 抵押品和对冲仓位持续再平衡

### stUSDe收益来源

- **资金费率**:永续合约的多空资金费率(通常多方支付)
- **质押收益**:stETH等生息资产的质押奖励
- **再平衡收益**:动态对冲过程中的交易收益

### 风险管理

- 分散至多家CEX降低交易对手风险
- 保险基金应对极端情况
- 实时监控和自动再平衡
- 储备金缓冲

## 经济模型

### USDe

- 1:1锚定美元
- 无固定供应上限
- 根据需求铸造和赎回
- 无发行费用,赎回可能有小额费用

### stUSDe

- 代表质押的USDe和累积收益
- 兑换率持续上涨
- 可随时解除质押(可能有冷却期)

### ENA代币(计划)

治理和激励代币:
- 协议治理权
- 收益分享
- 质押奖励
- 空投给早期用户(Shards积分计划)

## 应用场景

### 1. 稳定币持有

作为价值存储工具,替代USDT/USDC。

### 2. 高收益储蓄

质押stUSDe获得高于传统稳定币的收益。

### 3. DeFi抵押品

在Aave、Morpho等协议中作为抵押品借款。

### 4. 流动性提供

在DEX提供USDe/USDC等交易对流动性。

### 5. 杠杆收益策略

借入资产购买stUSDe,放大收益(同时放大风险)。

### 6. 套利

利用USDe锚定偏差进行套利。

## 风险与挑战

### 1. 资金费率风险

当市场转为空头主导,多方需支付空方资金费率,收益率下降甚至为负。

### 2. CEX交易对手风险

对冲仓位在中心化交易所,存在交易所倒闭或冻结风险。

### 3. 脱锚风险

极端市场波动或流动性危机可能导致USDe短期脱锚。

### 4. 监管风险

合成稳定币和衍生品策略可能面临监管审查。

### 5. 智能合约风险

尽管经过审计,仍存在潜在漏洞。

### 6. 可扩展性限制

受限于CEX永续合约的未平仓量和流动性。
