# Sky (原MakerDAO)

## 概念简介

Sky是DeFi生态中历史最悠久、影响最深远的去中心化稳定币协议,前身为MakerDAO。2024年8月,MakerDAO完成品牌升级,更名为Sky,同时推出了全新的代币体系和产品线。Sky协议的核心产品是去中心化稳定币USDS(原DAI)和治理代币SKY(原MKR),通过超额抵押机制保持稳定币与美元的1:1锚定。

MakerDAO由Rune Christensen于2014年创立,是以太坊上最早的DeFi协议之一,也是DeFi Summer的核心基础设施。Sky/MakerDAO不仅仅是稳定币发行商,更是一个去中心化的央行和信用体系,通过社区治理决定抵押品类型、稳定费率、风险参数等关键决策。

截至2024年,Sky协议管理着超过50亿美元的资产,USDS/DAI是仅次于USDT和USDC的第三大稳定币。协议在品牌重塑后进一步扩展产品线,推出了储蓄产品、代币化国债等创新业务。

## 核心特性

### 1. 双稳定币体系

- **USDS**:新品牌稳定币,改进的用户体验
- **DAI**:经典稳定币,继续支持
- 两者可1:1自由兑换
- DAI逐步过渡到USDS

### 2. 超额抵押机制

用户抵押加密资产铸造稳定币:
- 支持ETH、WBTC、stETH等主流资产
- 抵押率通常150%-200%
- 低于清算线会被清算
- 保证稳定币的偿付能力

### 3. Sky Savings Rate (SSR)

USDS持有者可获得储蓄利率:
- 无需锁定,随时存取
- 利率由治理决定(历史5-15%)
- 吸引用户持有USDS

### 4. Sky Token Stars

代币化美国国债产品:
- 链上持有真实世界资产
- 获得国债收益
- RWA业务的旗舰产品

### 5. 多抵押品支持

不同风险偏好的资产:
- 加密原生资产(ETH、WBTC)
- 流动性质押代币(stETH、rETH)
- 真实世界资产(RWA)
- 稳定币(USDC等)

### 6. 去中心化治理

SKY持有者决定协议运营:
- 调整稳定费率
- 添加新抵押品
- 分配协议收入
- 重大战略决策

## 技术优势

### 1. 去中心化

无需依赖单一实体或银行账户。

### 2. 透明性

所有抵押品和债务在链上公开可查。

### 3. 灵活性

支持多种抵押品类型,适应不同需求。

### 4. 可扩展性

通过PSM(锚定稳定模块)和RWA扩大规模。

### 5. 收益机会

为USDS持有者提供无风险储蓄收益。

## 发展历程

### MakerDAO时期(2014-2024)

**早期(2014-2017)**
- Rune Christensen创立MakerDAO
- 单抵押DAI(SAI)推出,仅支持ETH
- 发行MKR治理代币

**多抵押DAI(2019年11月)**
- 升级为多抵押品系统
- 支持ETH、BAT等多种资产
- 引入DSR(DAI储蓄利率)

**DeFi Summer(2020)**
- 成为DeFi基础设施核心
- DAI供应量爆发式增长
- 集成到数百个DeFi协议

**RWA探索(2021-2023)**
- 引入真实世界资产作为抵押品
- 与Centrifuge等RWA协议合作
- 投资美国国债和企业债券

**黑色星期四(2020年3月)**
- ETH暴跌导致大规模清算
- 系统债务和拍卖机制问题暴露
- 后续进行重大改进

**Endgame计划(2022-2023)**
- Rune提出Endgame长期规划
- 计划分拆为多个SubDAO
- 准备品牌重塑

### Sky时期(2024-)

**品牌升级(2024年8月)**
- MakerDAO更名为Sky
- MKR可升级为SKY(1 MKR = 24,000 SKY)
- DAI可升级为USDS(1:1)
- 推出全新视觉和产品线

**产品扩展**
- Sky Savings Rate储蓄产品
- Sky Token Stars代币化国债
- 改进用户体验和入口

## 核心产品

### USDS稳定币

- 去中心化、超额抵押的稳定币
- 与USD保持1:1锚定
- 可用于DeFi生态各类应用
- 兼容DAI,可自由兑换

### Sky Savings Rate

- USDS的储蓄账户
- 获得协议收入分成
- 无锁定期,灵活存取
- 类似传统银行的储蓄账户但链上化

### Sky Token Stars

- 代币化的美国国债投资产品
- 合规框架下的RWA
- 为用户提供稳定的美元收益
- 链上透明的真实世界资产敞口

### Vaults(金库)

用户开设金库抵押资产铸造USDS:
- 支持多种抵押品类型
- 灵活的抵押率和稳定费
- 自主管理债务头寸

## 经济模型

### SKY代币

- **总供应量**:约24亿枚(1 MKR = 24,000 SKY)
- **代币功能**:
  - 治理投票权
  - 协议收入分配(未来可能)
  - 系统债务的最后抵押品

### 收入来源

- 稳定费:铸造USDS的利息
- 清算罚金
- PSM费用
- RWA投资收益

### 收入分配

- 部分用于SSR支付
- 部分进入盈余缓冲区(系统储备)
- 未来可能分配给SKY持有者

## 应用场景

### 1. 去中心化稳定币

作为抗审查的美元替代品。

### 2. DeFi抵押品

USDS/DAI是DeFi中使用最广泛的稳定币之一。

### 3. 储蓄账户

通过SSR赚取被动收入。

### 4. 杠杆投资

抵押ETH铸造USDS,再购买更多ETH(循环借贷)。

### 5. 企业和DAO资金管理

作为去中心化的美元储备。

## 风险与挑战

### 1. 抵押品风险

加密资产价格波动可能导致清算。

### 2. RWA风险

真实世界资产引入托管和法律风险。

### 3. 治理风险

决策可能受大户或特定利益集团影响。

### 4. 竞争压力

USDT、USDC等中心化稳定币占据主导地位。

### 5. 监管不确定性

RWA和稳定币业务面临监管挑战。

### 6. 品牌过渡

从MakerDAO到Sky的品牌转换需要时间和用户接受。

## 竞争对手

- **USDT/USDC**:中心化稳定币,规模最大
- **Frax Finance**:部分抵押+算法稳定币
- **Liquity**:无治理的去中心化稳定币(LUSD)
- **Aave GHO**:Aave原生稳定币
- **Curve crvUSD**:Curve的LLAMA算法稳定币

## Sky/MakerDAO的优势

- DeFi领域最悠久的信誉
- 最去中心化的稳定币治理
- 成熟的风险管理框架
- 丰富的RWA业务经验
- 广泛的DeFi集成

## 最佳实践

### USDS持有者

- 考虑存入SSR赚取储蓄利率
- 关注协议治理决策

- 了解不同抵押品的风险
- 定期检查稳定费率变化

### SKY持有者

- 积极参与治理投票
- 了解协议长期规划
- 关注收入和盈余缓冲区健康度
