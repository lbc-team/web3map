## Backpack

Backpack 是革命性的 Web3 钱包，内置 xNFT（可执行 NFT）运行时，允许用户在钱包内直接运行 DApp。

### 核心特性

**1. xNFT 运行时**
- NFT 可以是可执行程序
- 在钱包内运行完整 DApp
- 使用 React 开发

**2. 多链支持**
- [Solana](https://learnblockchain.cn/tags/Solana?map=Solana), Ethereum, Polygon

**3. 应用商店**
- 内置 xNFT 应用商店
- DeFi、NFT、游戏等应用

**4. 社交功能**
- 用户名系统
- 好友列表和消息

### xNFT 技术

**什么是 xNFT**
xNFT = NFT + 可执行代码：
- 元数据在链上
- 代码托管在 IPFS
- 在 Backpack 中执行

### 与传统钱包对比

| 特性 | Backpack | [Phantom](https://learnblockchain.cn/tags/Phantom?map=Solana) |
|------|----------|---------|
| xNFT | 支持 | 不支持 |
| 应用商店 | 有 | 无 |
| DApp 运行 | 钱包内 | 浏览器 |

### 愿景

Backpack 的目标是成为 Web3 的"应用商店"，让钱包成为应用平台。

### 相关概念

- **xNFT**：可执行 NFT
- **[Phantom](https://learnblockchain.cn/tags/Phantom?map=Solana)**：传统钱包
- **MetaMask Snaps**：类似概念
