## Orb

Orb 是 Helius 开发的新一代 [Solana](https://learnblockchain.cn/tags/Solana?map=Solana) 区块链浏览器，专注于提供更好的用户体验和更强大的数据分析功能。

官网：https://orb.helius.dev/

### 核心特点

**现代化设计**
- 简洁美观的界面
- 响应式设计
- 流畅的交互体验

**强大的搜索**
- 交易签名查询
- 账户地址查询
- 程序查询
- Token 查询
- NFT 查询

**实时数据**
- 最新交易流
- 实时网络统计
- 区块生产监控

### 主要功能

**1. 交易解析**
人类可读的交易展示：
- 自动识别交易类型（转账、Swap、NFT 交易等）
- 清晰的资金流向
- 程序调用栈
- 日志信息

**2. 账户分析**
全面的账户信息：
- Token 余额列表
- NFT 持有
- 交易历史
- 程序所有权

**3. 程序浏览**
智能合约信息：
- 程序 ID
- 部署信息
- 调用统计
- 相关账户

**4. Token 详情**
代币全面数据：
- 供应量
- 持有者分布
- 价格走势（如果有）
- 转账活动

**5. NFT 展示**
NFT 集合和个体：
- 元数据展示
- 交易历史
- 地板价
- 持有者

### 技术优势

**Helius 基础设施**
基于 Helius 的强大 RPC 和数据服务：
- 快速响应
- 完整数据
- cNFT 支持
- 增强 API

**实时索引**
通过 Geyser 实时索引链上数据：
- 零延迟更新
- 完整历史记录
- 深度分析

**智能解析**
自动识别和解析：
- DeFi 操作（Swap、流动性提供等）
- NFT 交易（Mint、Sale、Transfer）
- 质押操作
- 跨程序调用

### 用户体验

**快速加载**
优化的性能，秒级加载：
- CDN 加速
- 缓存优化
- 渐进式加载

**移动友好**
完美适配移动端：
- 响应式布局
- 触控优化

**深色模式**
支持深色/浅色主题切换。
