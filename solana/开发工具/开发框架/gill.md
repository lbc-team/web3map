## gill

gill 是基于 Kit 开发的 Solana 客户端和开发工具，专注于提供更简洁的开发体验和更高的性能。

### 核心特点

**基于 Kit**
gill 构建在 Anza（原 Solana Labs）的 Kit 工具集之上：
- 继承 Kit 的最佳实践
- 模块化架构
- 易于扩展

**开发友好**
- 简化的 API
- 清晰的文档
- 快速上手

**性能优化**
- 优化的数据结构
- 高效的网络通信
- 降低资源消耗

### 使用场景

**客户端开发**
作为 Solana 验证器客户端的轻量级替代。

**开发工具**
为开发者提供辅助工具和库。

**研究和实验**
探索新的客户端实现方式。

### 与其他客户端对比

| 特性 | gill | Agave | Sig |
|------|------|-------|-----|
| 基础 | Kit | 独立 | 独立 |
| 语言 | Rust | Rust | Zig |
| 成熟度 | 早期 | 成熟 | 开发中 |
| 目标 | 简洁 | 全功能 | 性能 |

### 开发状态

gill 目前处于早期开发阶段：
- 积极开发中
- 功能逐步完善
- 欢迎社区贡献
