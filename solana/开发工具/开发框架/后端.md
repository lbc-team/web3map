## 后端

[Solana](https://learnblockchain.cn/tags/Solana?map=Solana) 后端开发指构建链下服务与 [Solana](https://learnblockchain.cn/tags/Solana?map=Solana) 交互，如索引、监控、自动化等。

### 使用场景

**数据索引**
通过 Geyser 或 RPC 获取链上数据，存储到数据库。

**自动化任务**
- 清算机器人
- 套利机器人
- 自动做市

**API 服务**
为前端提供聚合数据和业务逻辑。

### 技术选择

**Node.js**
使用 `@solana/web3.js`

**[Rust](https://learnblockchain.cn/tags/Rust)**
使用 `solana-client`，性能最优

**Python**
使用 `solana-py`，适合数据分析

**[Go](https://learnblockchain.cn/tags/Go)**
使用 `solana-go`，适合高并发服务

### 典型架构

```
Solana 区块链
    ↓
Geyser Plugin (实时数据)
    ↓
后端服务 (索引/处理)
    ↓
数据库 (PostgreSQL/Redis)
    ↓
API (提供给前端)
```

### 最佳实践

- 使用专业 RPC 服务（[Helius](https://learnblockchain.cn/tags/Helius?map=Solana), QuickNode）
- 实现重试和错误处理
- 缓存频繁查询的数据
- 监控和日志

### 相关概念

- **Geyser**：实时数据流
- **RPC**：区块链查询接口
- **索引**：链上数据结构化存储
