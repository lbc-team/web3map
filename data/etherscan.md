# Etherscan

## 概念简介

Etherscan 是以太坊区块链的官方区块浏览器和分析平台，由 Etherscan.io 团队于2015年创建。作为以太坊生态中最知名、使用最广泛的区块浏览器，Etherscan 为全球数百万用户提供以太坊链上数据的查询、追踪和分析服务。

Etherscan 不仅是一个简单的区块浏览器，更是以太坊生态的基础设施。它提供交易查询、智能合约验证、代币追踪、Gas 追踪器、DApp 目录等丰富功能。几乎每个以太坊用户都会使用 Etherscan 查询交易状态、验证合约代码、查看钱包余额。Etherscan 已经成为以太坊透明性和可信度的重要保障。

## 核心特性

### 区块链数据查询
Etherscan 提供全面的以太坊链上数据查询：
- **交易查询**：输入交易哈希查看交易详情、状态、Gas 费、事件日志
- **区块查询**：查看区块信息、包含的交易列表、矿工/验证者
- **地址查询**：地址余额、ETH 和代币交易历史、合约代码
- **代币追踪**：ERC20、ERC721、ERC1155 代币的转账记录和持有者
- **批量查询**：支持批量查询多个地址或交易

### 智能合约验证
Etherscan 是智能合约验证的权威平台：
- **源码验证**：开发者可以上传合约源代码进行验证
- **代码查看**：验证后的合约可以在 Etherscan 上直接查看源码
- **合约交互**：通过 Web 界面直接调用合约的读/写方法
- **代理合约识别**：自动识别代理合约模式，显示实现合约
- **安全审计信息**：显示合约是否经过审计

合约验证极大提升了透明度，用户可以验证合约是否与项目声称的一致。

### Gas 追踪器
实时 Gas 价格监控工具：
- **实时 Gas 价格**：Low、Average、High 三档 Gas 价格建议
- **Gas 价格历史**：历史 Gas 价格趋势图表
- **Gas 消耗排名**：显示消耗 Gas 最多的合约和交易
- **Gas 预估工具**：估算不同类型交易的 Gas 成本

Gas 追踪器帮助用户选择合适的交易时机，节省 Gas 费用。

### 代币和 NFT 追踪
- **代币信息**：代币的总供应量、持有者数量、价格、交易量
- **代币持有者排名**：代币富豪榜
- **NFT 追踪**：NFT 的元数据、交易历史、持有者信息
- **代币批准管理**：查看和撤销代币授权（Token Approval）

### DApp 和项目目录
- **DApp 目录**：收录热门 DApp，按类别分类（DeFi、DEX、游戏等）
- **项目信息**：项目的官方链接、社交媒体、合约地址
- **验证标记**：为官方合约添加验证标记，防止钓鱼

### 账户功能
注册账户后可以使用高级功能：
- **地址监控**：监控特定地址的交易活动
- **交易备注**：为自己的交易添加私有备注
- **API 密钥**：获取 API 访问密钥
- **自定义设置**：个性化界面设置

### Etherscan API
为开发者提供强大的 API 服务：
- **免费套餐**：每秒5次请求的免费 API 访问
- **账户 API**：查询地址余额、交易历史
- **合约 API**：获取合约 ABI、源码
- **交易 API**：查询交易状态和收据
- **区块 API**：获取区块数据
- **事件日志 API**：查询合约事件
- **Gas Tracker API**：获取 Gas 价格数据
- **代币 API**：代币余额和转账查询

## 平台特色

**权威性和可信度**：作为以太坊生态中最早、最知名的区块浏览器，Etherscan 已经成为事实上的以太坊数据权威来源。项目方都会在 Etherscan 上验证合约以获得可信度。

**数据完整性**：索引了以太坊从创世区块至今的所有数据，提供完整的历史数据查询能力。数据准确可靠，是司法、审计等场景的重要证据来源。

**用户体验**：经过多年迭代，Etherscan 拥有直观友好的用户界面。即使是非技术用户也能轻松使用。

**功能丰富**：不断推出新功能，如 NFT 追踪、代币批准管理、合约交互界面等，满足用户多样化需求。

**API 生态**：Etherscan API 被数千个项目和工具使用，是 Web3 基础设施的关键组成部分。

**多网络支持**：除了以太坊主网，Etherscan 团队还运营着测试网浏览器（Goerli、Sepolia）和兼容链浏览器（BSCScan、PolygonScan、Arbiscan 等）。

**社区信任**：经过8年的运营，Etherscan 赢得了以太坊社区的广泛信任，是生态中不可或缺的基础设施。

## 发展历程

**2015年**：Etherscan 正式上线，为以太坊提供区块浏览器服务。

**2016-2017年**：随着以太坊生态的发展，Etherscan 添加了代币追踪、合约验证等功能。成为以太坊用户的必备工具。

**2018-2019年**：推出 API 服务，为开发者提供数据接口。扩展功能，增加 Gas 追踪器、DApp 目录等。

**2020-2021年**：DeFi 和 NFT 爆发，Etherscan 增加相关的追踪和分析功能。推出代币批准管理工具，帮助用户保护资产安全。

**2022年**：支持以太坊合并（The Merge），适配 PoS 共识机制。推出更多面向用户的安全功能。

**2023年**：继续优化用户体验，推出新的数据分析工具。团队扩展到其他 EVM 链，运营多个链的浏览器。

**2024年至今**：作为以太坊生态的核心基础设施，Etherscan 持续创新，提供更好的服务。

## 应用场景

**普通用户**：查询交易状态、查看钱包余额、监控转账记录。

**开发者**：调试智能合约、查看事件日志、验证合约代码、集成 API。

**投资者**：追踪巨鲸地址、查看代币持有者分布、监控项目方资金流动。

**安全研究**：分析黑客攻击、追踪被盗资金、识别钓鱼合约。

**审计和合规**：提供链上数据证明、生成交易报告、税务计算。

**项目方**：验证官方合约、展示项目透明度、与社区沟通。

**媒体和研究**：获取准确的链上数据用于报道和研究。

**教育**：学习以太坊工作原理、理解智能合约执行过程。
